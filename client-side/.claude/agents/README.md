# Agent Roles

Use these role files as prompts for Claude subagents.

- `main-agent.md`: orchestrates work and merges decisions
- `system-design.md`: architecture, data flow, state, routes
- `ui.md`: visual and responsive implementation
- `bug-fix.md`: reproduce, isolate, patch, verify
- `integration.md`: connect UI, Redux, API, and routing
- `templates.md`: ready-to-use prompts for fast agent handoff
